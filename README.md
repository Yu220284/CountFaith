# CountFaith

偽造薬を専門家の目で防ぐ、医薬品流通監査システム

## 概要

CountFaithは、正規の医薬品流通経路において、AIが検知したリスク地点や抜き打ち検査が必要なポイントへ、地域に眠る「潜在薬剤師」を「監査官（Supervisor）」として派遣するプラットフォームです。

## 三者のエコシステム

| ステークホルダー | 役割 | メリット |
|---|---|---|
| 政府・製薬会社 | 依頼主。監査の実行指示と報酬の支払い | 流通の信頼性確保、偽造薬による経済・健康被害の防止 |
| 潜在薬剤師 | 現場監督者。流通拠点に赴き、真贋判定と管理体制をチェック | 専門資格の活用、柔軟な働き方での高額報酬、社会的貢献 |
| 流通拠点 | 被監査者。薬剤師の入室とデータ照合を許可 | CountFaith認定による信頼獲得、不正混入リスクの排除 |

## 技術スタック

- **Crustdata**: ターゲットとなる流通拠点の周辺に住む潜在薬剤師をリストアップ
- **Blaxel**: 薬剤師の専門経歴と拠点までの距離を計算し、最適な人物に「クエスト（派遣依頼）」を自動送信。Proof of Faith（真実証明）を発行

## 開発

```bash
npm install
npm run dev
```

http://localhost:9002 でアプリケーションが起動します。

## 主要機能

- **ダッシュボード**: 監査状況の可視化
- **薬剤師管理**: 登録薬剤師の一覧と管理
- **監査チェック**: リスク拠点の選択、薬剤師の自動割当、Proof of Faith発行
