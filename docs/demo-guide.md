# CountFaith - 実装状況とデモシナリオ

## ✅ 実装完了機能

### 1. **Crustdata API統合** ⭐ 核心機能
- **地理的検索**: 拠点から50km以内の薬剤師を自動検索
- **実データ取得**: 実際のLinkedInプロフィールから薬剤師情報を取得
- **経験年数フィルター**: 3年以上の経験を持つ薬剤師のみ表示
- **リアルタイム検索**: 各拠点選択時に即座に周辺の薬剤師を表示

### 2. **ダッシュボード**
- リスク拠点の可視化
- 監査状況の統計表示
- グローバルマップビュー

### 3. **監査チェック作成フロー**
- ① リスク拠点選択
- ② 周辺薬剤師の自動検出（Crustdata）
- ③ 最適な薬剤師への割当（Blaxel - モック）
- ④ Proof of Faith発行（Blaxel - モック）

### 4. **薬剤師管理**
- 登録薬剤師一覧
- 稼働状況の可視化
